# Unveiling the Android App Market: Analyzing Google Play Store Data

## Objective
This project analyzes the Google Play Store dataset to understand app market dynamics, including category distribution, ratings, pricing, and user sentiment, in order to derive actionable business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features
- App name, Category, Rating, Reviews, Size, Installs
- Type (Free/Paid), Price, Content Rating, Genres
- Last Updated, Current Version, Android Version
- User reviews with pre-labeled Sentiment, Sentiment Polarity, and Subjectivity

## Analysis Performed
- Data Cleaning (Installs, Price formatting; missing value handling)
- Category Distribution Analysis
- Metrics Analysis (Ratings, Size, Installs, Free vs Paid)
- Sentiment Analysis using user review data
- Correlation between Sentiment and App Rating
- Data Visualization (bar charts, histograms, scatter plot)

## Key Insights
- FAMILY and GAME are the most saturated categories on the Play Store.
- COMMUNICATION apps have the highest average installs despite not being the most numerous category.
- Most apps are Free, with the majority maintaining ratings between 4.0 and 4.5.
- Around 64% of user reviews are positive, with a mild positive relationship between sentiment and app rating.

## Conclusion
This analysis reveals clear patterns in category saturation, install popularity, and user sentiment across the Google Play Store, with Communication and Gaming apps driving the highest engagement and overall sentiment remaining largely positive.
