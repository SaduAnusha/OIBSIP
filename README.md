# OIBSIP - Oasis Infobyte Data Analytics Internship

Data Analytics internship projects showcasing exploratory data analysis, customer segmentation, data visualization, and business insights using Python.

## Projects

### [Task 1: EDA on Retail Sales Data](./Task1_Retail_Sales_EDA)
Exploratory data analysis on retail transaction data to identify sales trends, customer purchasing behavior, and product performance.

### [Task 2: Customer Segmentation Analysis](./Task2_Customer_Segmentation)
Segmented customers into distinct groups using K-Means clustering based on income, age, and spending behavior, to support targeted marketing strategies.

### [Task 3: Unveiling the Android App Market](./Task3_Android_App_Market)
Analyzed Google Play Store data to understand app category trends, ratings, pricing, and user review sentiment.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook
