*# EDA on Retail Sales Data*



*## Objective*

*The objective of this project is to analyze retail sales transaction data to identify sales trends, customer purchasing behavior, and product performance using Exploratory Data Analysis (EDA).*



*## Tools Used*

*- Python*

*- Pandas*

*- NumPy*

*- Matplotlib*

*- Seaborn*

*- Jupyter Notebook*



*## Dataset Features*

*- Transaction ID*

*- Date*

*- Customer ID*

*- Gender*

*- Age*

*- Product Category*

*- Quantity*

*- Price per Unit*

*- Total Amount*



*## Analysis Performed*

*- Data Cleaning*

*- Missing Value Analysis*

*- Duplicate Value Analysis*

*- Descriptive Statistics*

*- Product Category Analysis*

*- Gender Analysis*

*- Age Group Analysis*

*- Monthly Sales Trend Analysis*

*- Correlation Analysis*



*## Key Insights*

*- Female customers generated slightly higher revenue than male customers.*

*- Product category performance varied across categories.*

*- Customer spending patterns differed across age groups.*

*- Monthly sales trends revealed fluctuations in purchasing behavior.*

*- Higher purchase quantities contributed to greater transaction amounts.*



*## Conclusion*

*The analysis provides valuable insights into customer behavior and sales performance, supporting data-driven business decisions.*
